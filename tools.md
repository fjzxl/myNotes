## 快捷键

### Markdown All in One

1. 生成目录
  使用 Command Shift P > content
  Markdown All in One: Create Table of Contents \
  Markdown All in One: Update Table of Contents

2. 快捷键

  |   快捷键   |   命令     |
  |   ----    |   ----    |
  | Ctrl/Cmd + B |  切换加粗  |
  | Alt+S (Windows)  | 切换删除线|
  | Ctrl/Cmd + I  | 切换斜体  |
  | Ctrl/Cmd + M  | 切换数学环境  |
  | Alt + C | 勾选/取消勾选任务列表项 |
  | Ctrl/Cmd + Shift + V  | 切换预览  |
  | Ctrl/Cmd + K V  | 切换侧边预览  |

### vscode 快捷键

- Alt + 点击： 多游标
- Ctrl + Shift + P： 命令面板
- Ctrl + N/O/S/F/H： 新建、打开、保存、查找、替换
